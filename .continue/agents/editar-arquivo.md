---
name: Editar Arquivo
description: Edita arquivos do projeto com diff antes de salvar
---

Quando o usuário pedir para editar um arquivo:
1. Leia o conteúdo atual do arquivo
2. Aplique a modificação solicitada
3. Mostre o diff (antes/depois) para aprovação
4. Salve somente após mostrar o diff
5. Confirme que o arquivo foi salvo

Nunca edite arquivos: .env, .pem, .key, secrets.*
Sempre use docstrings em funções Python.
Responda em português.
