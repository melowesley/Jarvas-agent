🚀 Bem-vindo ao Jarvas

Jarvas é um assistente de IA distribuído com múltiplos guardas especializados, roteamento inteligente de modelos e mineração automática de progresso. Em vez de depender de um único modelo, o Jarvas roteia cada mensagem para o modelo mais adequado com base no tipo de tarefa, expõe guardas especializados (Gemini e DeepSeek) para análises específicas e minera automaticamente o conhecimento de conversas encerradas.

O que você pode fazer?

💬 Chat Inteligente

Faça perguntas e o Jarvas automaticamente detecta o tipo de tarefa e seleciona o modelo mais apropriado.

🎯 Slash Commands

Use comandos especializados como /g (Gemini), /d (DeepSeek), /debate e /hmem (MemPalace).

🧠 Memória Semântica

O Jarvas apre automaticamente com cada conversa, armazenando aprendizados em seu MemPalace.

📊 API REST

Integre Jarvas em suas aplicações usando a API REST com suporte a Server-Sent Events.

🤝 Debate Automático

Veja diferentes perspectivas: Gemini vs DeepSeek debatem tópicos e chegam a um consenso.

⚙️ Agentes Customizáveis

Crie agentes gerenciáveis com sessões isoladas, histórico persistente e ferramentas customizadas.

Arquitetura Geral

Jarvas 0.3.0
Framework multi-agente com roteamento automático, mineração de conhecimento em tempo real e persistência em Supabase + MemPalace. 
CLI/API → Router → Modelo Selecionado
                    ↓
          Guardas Especializados (Gemini/DeepSeek)
                    ↓
              Mineradores (Background)
                    ↓
        MemPalace + Supabase (Persistência)

📋 Setup & Configuração

Instalação

Certifique-se de ter Python 3.11+ instalado e clone o repositório:

git clone https://github.com/melowesley/Jarvas-agent.git
cd jarvas
pip install -r requirements.txt

Variáveis de Ambiente

Crie um arquivo .env na raiz do projeto com as seguintes variáveis:

# APIs Externas
OPENROUTER_API_KEY=seu_openrouter_key
GEMINI_API_KEY=seu_gemini_key
DEEPSEEK_API_KEY=seu_deepseek_key

# Supabase (Persistência)
SUPABASE_URL=sua_supabase_url
SUPABASE_KEY=sua_supabase_key

# MemPalace (Memória Semântica)
MEMPALACE_URL=http://localhost:8000  # ou sua URL de MemPalace

# Ollama (Opcional - para modelos locais)
OLLAMA_BASE_URL=http://localhost:11434

Obtendo as Chaves

OpenRouter API ▼
  1. Acesse https://openrouter.ai
  2. Crie uma conta ou faça login
  3. Vá para Dashboard → Keys
  4. Crie uma nova chave e copie
  5. Adicione créditos à sua conta para usar os modelos
Google Gemini API ▼
  1. Acesse https://ai.google.dev
  2. Clique em "Get API Key"
  3. Crie um novo projeto ou selecione existente
  4. Copie a chave gerada
DeepSeek API ▼
  1. Acesse https://platform.deepseek.com
  2. Faça login ou crie conta
  3. Vá para API Keys
  4. Crie uma nova chave

Verificando a Instalação

python -m jarvas --help
✅ Pronto! Se você vir as opções de ajuda, a instalação foi bem-sucedida.

💻 CLI Interativa

Iniciando a CLI

A forma mais simples de usar o Jarvas é através da CLI interativa:

jarvas

Você verá um prompt esperando sua entrada:

você > _

Três Formas de Usar

1. Pergunta Direta

você > Como eu funciono?

O Jarvas processa sua pergunta, detecta o tipo de tarefa, seleciona o modelo apropriado e responde.

2. Conversa Contínua

você > Explique roteamento de IA
jarvas > Roteamento de IA é o processo de direcionar...

você > E qual é a vantagem disso?
jarvas > A principal vantagem é...

Mantenha o histórico da conversa durante a sessão.

3. Retomar Contexto Anterior

jarvas continuar ontem 15h

Retoma a conversa do dia anterior às 15h (se disponível em Supabase).

Atalhos e Comandos Especiais

Comando Descrição
/help Mostra lista de slash commands disponíveis
/clear Limpa o histórico da sessão atual
/exit ou Ctrl+D Sair da CLI
/history Mostra histórico de mensagens da sessão

Detecção Automática de Tipo de Tarefa

O Jarvas detecta automaticamente o tipo de tarefa e seleciona o melhor modelo:

📊 Analysis

Detectado: Perguntas sobre dados, estatísticas, insights
Modelo: Claude 3.5 Sonnet

💻 Code

Detectado: Perguntas sobre programação, debugging, arquitetura
Modelo: Llama 3.3 70B

🖼️ Vision

Detectado: Análise de imagens, OCR, reconhecimento visual
Modelo: GPT-4 Vision

💬 Chat

Detectado: Conversas gerais, perguntas abertas
Modelo: Hermes 3

⚡ Slash Commands

Slash commands permitem você acessar guardas especializados e funcionalidades avançadas do Jarvas. Use /help para ver todos os comandos disponíveis.

Guardas Diretos

/g <prompt> — Gemini Direto ▼

Força uma resposta do Gemini 2.5 Flash (análise, web search, busca de informações).

Exemplo: 
você > /g Qual é a capital da Austrália?
gemini > A capital da Austrália é Canberra...

Análise Busca Pesquisa

/g web <busca> — Gemini + Web Search ▼

Usa Gemini com busca web em tempo real para informações atualizadas.

Exemplo: 
você > /g web últimas notícias sobre IA em 2026
gemini > [Busca a web] As últimas notícias sobre IA incluem...

Web Search Real-time

/d <prompt> — DeepSeek Direto ▼

Força uma resposta do DeepSeek (estruturação, análise de código, lógica).

Exemplo: 
você > /d Como implementar um algoritmo de ordenação eficiente?
deepseek > Existem vários algoritmos: MergeSort, QuickSort, HeapSort...

Código Estruturação

/d web <busca> — DeepSeek + Web Search ▼

DeepSeek com busca web para análises estruturadas de informações recentes.

Exemplo: 
você > /d web tendências de tecnologia 2026
deepseek > [Busca a web] As tendências incluem: 1) IA Distribuída, 2) Quantum Computing...

Debate Multi-Agente

/debate <tópico> — Debate Gemini vs DeepSeek ▼

Veja dois agentes especializados debatem um tópico e chegam a um consenso.

Exemplo: 
você > /debate IA generativa é perigosa?

[Rodada 1] Gemini: A IA generativa pode ser perigosa se...
[Rodada 2] DeepSeek: Concordo que há riscos, mas também há benefícios...
[Rodada 3] Gemini: Isso faz sentido. O consenso é que...

3 Rodadas Consenso

Forçar Modelo Específico

/hopen <model-id> — Força Modelo ▼

Bypassa o roteamento automático e força um modelo específico do OpenRouter.

Modelos disponíveis:
  • llama-3.3-70b - Llama 3.3 (código e lógica)
  • hermes-3-extended - Hermes 3 (chat geral)
  • claude-3.5-sonnet - Claude Sonnet (análise)
  • gpt-4-vision - GPT-4 Vision (imagens)
Exemplo: 
você > /hopen llama-3.3-70b
você > Agora escreva uma função recursiva em Python

MemPalace (Memória Semântica)

/hmem status — Status do MemPalace ▼

Mostra o status atual do sistema de memória.

você > /hmem status
MemPalace > Wings: 2
            Drawers: 42
            Total embeddings: 1247
/hmem list — Listar Wings (Categorias) ▼

Lista todas as categorias (wings) de memória.

você > /hmem list
MemPalace > Wings disponíveis:
            - jarvas/learnings (aprendizados)
            - jarvas/code (snippets)
/hmem search <busca> — Busca Semântica ▼

Busca aprendizados relacionados usando semântica (não keyword matching).

você > /hmem search problema com async em Python
MemPalace > Encontrado 3 drawers relacionados:
            1. [confidence: 0.94] Como usar asyncio...
            2. [confidence: 0.87] Debugging de coroutines...
            3. [confidence: 0.71] Event loops explicado
/hmem add <wing> <room> <conteúdo> — Adicionar Memória ▼

Adiciona manualmente um aprendizado ao MemPalace.

Exemplo: 
você > /hmem add jarvas/learnings async "useEffect sem dependências causa loops"
MemPalace > ✅ Drawer adicionado com sucesso (ID: d42c8f2a)
/hmem get <id> — Obter Drawer Específico ▼

Recupera um aprendizado específico pelo ID.

você > /hmem get d42c8f2a
MemPalace > ID: d42c8f2a
            Conteúdo: useEffect sem dependências causa loops
            Criado: 2026-04-10 14:32
            Confidence: 0.92
/hmem del <id> — Deletar Drawer ▼

Remove um aprendizado do MemPalace.

você > /hmem del d42c8f2a
MemPalace > ✅ Drawer deletado

Agentes Gerenciáveis

/session list — Listar Agentes ▼

Lista todos os agentes disponíveis para criar sessões.

você > /session list
Agentes disponíveis:
- hermes (Chat geral)
- gemini-guard (Análise especializada)
- deepseek-guard (Estruturação)
- gemma-local (Agente local via Ollama)
/session new <agent> — Criar Sessão ▼

Cria uma nova sessão com um agente específico.

você > /session new gemini-guard
✅ Sessão criada: sess_a8f2k9d
Conectado ao agente: gemini-guard

você > Sua pergunta aqui?
/session send <id> <msg> — Enviar Mensagem ▼

Envia uma mensagem para uma sessão específica.

você > /session send sess_a8f2k9d Qual é a capital do Brasil?
Agente (sess_a8f2k9d) > A capital do Brasil é Brasília...
/session history <id> — Ver Histórico ▼

Mostra o histórico completo de uma sessão.

você > /session history sess_a8f2k9d
[Mensagem 1] você: Qual é a capital do Brasil?
[Mensagem 2] agente: A capital do Brasil é Brasília...
[Mensagem 3] você: Quantos habitantes tem?
[Mensagem 4] agente: Brasília tem aproximadamente 3.1 milhões...
